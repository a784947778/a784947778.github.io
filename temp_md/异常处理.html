异常处理

# Springboot项目

# @value不能获取yml自定义变量的问题

解决方法：

key:value中的value值和冒号之间要有空格 列：key: value，要严格按照格式来

# es和kibana

# Error: listen EADDRNOTAVAIL: address not available 114.96.78.165:9280

解决方法：

云服务器，kibana.yml中的server.host 要写内网ip或者0.0.0.0

# Unable to retrieve version information from Elasticsearch nodes

解决办法：

网络问题，不能用localhost，修改kibana.yml中的elasticsearch.hosts为云服务器的公网ip

# Error: Unable to write Kibana UUID file, please check the uuid.server configuration value in kibana.yml and ensure Kibana has sufficient permissions to read / write to this file. Error was: EACCE

解决方法：

# 在root下为 为kibana赋权

chown -R elsearch:elsearch /home/es/kibana-7.6.1-linux-x86_64/

# 修改kibana所在文件夹的权限

chmod 770 elsearch:elsearch /home/es/kibana-7.6.1-linux-x86_64

# redis

# MISCONF Redis is configured to save RDB snapshots, but is currently not able to persist on disk. Com；

解决方法：

文件权限问题：redis Failed opening .rdb for saving: Read-only file system，修改dump.rdb的权限为777
